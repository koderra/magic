STYLE GUIDE

Reserved words

All components are implemented with "base" style, and they change styling including colour by adding any of the reserved words. If you modify a style with any of those reserved words, it may affect the look and feel of a component. This is intentional because it allows for shared global level custom styles

  • info
  • error
  • warning
  • default
  • base
  • success
  • highlight
  • alert
